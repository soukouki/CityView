# Simutrans CityView

Simutrans CityViewとは、Simutransのゲーム画面のスクリーンショットを利用し、Web上の地図として閲覧できるようにするツールです。

## 機能

- 開発中
  - 各セーブデータのマップ閲覧
- 計画中
  - 年代ごとの切り替え表示
  - NSマップのライブビュー
  - マップ全体を写した高解像度画像生成
  - 駅名などによる検索機能
  - マップの見どころスポットの表示
